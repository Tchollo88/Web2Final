# Web2Final

A web application and API project developed as a final project for Web Development II coursework in the AAS Application Development program.

## Technologies Used

* C#
* ASP.NET Web API
* Entity Framework
* SQL Server / Database-backed models
* Visual Studio
* Git / GitHub

## Features

* Customer management
* Product management
* Order management
* Database-backed data storage
* CRUD operations (Create, Read, Update, Delete)
* REST-style API endpoints
* Entity Framework data access

## Skills Demonstrated

* Object-oriented programming
* Web API development
* Database design and management
* Entity Framework usage
* Data modeling
* CRUD implementation
* Application troubleshooting and debugging

## Project Structure

* `Controllers` – API endpoints and application logic
* `Models` – Customer, Product, and Order data models
* `Data` – Database context and configuration
* `Migrations` – Database schema migrations

## Purpose

This project was created to demonstrate web application development concepts, API design, database interaction, and Entity Framework integration. The application manages customer, order, and product data through database-backed models and API endpoints.

## What I Learned

Through this project I gained experience working with:

* ASP.NET Web API architecture
* Entity Framework and database connectivity
* CRUD operations
* Structured application development
* Debugging and troubleshooting data-driven applications
